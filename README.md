# Shoe Store - Информационная система магазина обуви

Проект разрабатывается для ООО «Обувь» в рамках демонстрационного экзамена.

## Роли пользователей
- Гость
- Авторизованный клиент
- Менеджер
- Администратор

## Функциональные модули
- Авторизация
- Просмотр товаров
- Управление товарами (админ)
- Управление заказами (админ/менеджер)

## Технологии
- Бэкенд: Python (Flask/Django)
- База данных: Supabase (PostgreSQL)
- Фронтенд: HTML, CSS
